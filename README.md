# 📱 LINE Expense Bot

บอทสำหรับบันทึกค่าใช้จ่ายผ่าน LINE  
พิมพ์ในแชทหรือกลุ่ม เช่น:

!ex: กาแฟ 95 บาท เมื่อวาน

yaml
คัดลอกโค้ด

บอทจะจัดหมวดหมู่ + เก็บข้อมูลลง **Supabase** อัตโนมัติ
